# HIGHRISE-Encrypt-Decrypt
Secure Remote Communication Using DES Encryption Algorithm

The system was created using the following languages. The system front end was created using HTML and CSS while the backend using PHP.
The application allows a sender to send an encrypted message to a recipient and also for the recipient to decrypt the sent message.
The encrypted message is sent to the recipient's email address while the decryption key is sent to the recipient's messages.
The recipient will use our application to decrypt the message sent. For this purpose, he/she will input the encrypted message and the decryption key.
The decryption key is an alphanumeric 10-digit long sent and used for a specific encrypted message.
The algorithm was implemented using PHP.
The email is sent through PHP mailer while the SMS is sent through MessageBird API.
